# 四大核心领域定义

在采集和筛选时，用以下定义判断相关性。

---

## 领域 A： AIGC / 生成式AI

**核心关键词**：
```
text-to-video, image generation, text-to-image,
文生视频, 文生图, 图生视频,
Sora, Seedance, Kling, Runway, 即梦, 可灵,
Llama, GPT, Claude, Gemini, 开源模型,
diffusion, transformer architecture,
multimodal, reasoning model
```

**关注重点**：
- 新模型发布（技术突破 + 产品落地）
- 生成质量 benchmark 突破
- 开源工具 / weights 发布
- API 价格变动
- 融资并购事件

---

## 领域 B： 虚拟数字人

**核心关键词**：
```
digital human, talking head, avatar,
数字人, 虚拟主播, 数字分身, 虚拟偶像,
neural rendering, 3D avatar, photorealistic,
face swap, voice clone, lip sync,
虚拟形象, AI主播, 数字员工
```

**关注重点**：
- 驱动技术突破（表情/动作/声音合成）
- 实时交互能力进展
- 商业落地案例（直播/客服/教育）
- 平台政策与监管动向

---

## 领域 C： AI短视频 / 短剧

**核心关键词**：
```
AI editing, short video, 剪映AI,
AI漫剧, AI短剧, AI动画, 视频生成工作流,
video editing, content creation tool,
YouTube Shorts, TikTok, 短视频创作,
automated editing, AI剪辑, storyboard
```

**关注重点**：
- 制作工具更新（新功能/价格调整）
- 工作流优化（效率提升工具）
- 产业数据（用户规模/市场数据）
- 平台算法变化
- 热门内容案例分析

---

## 领域 D： 智能传播 / 舆情

**核心关键词**：
```
AI journalism, computational propaganda,
舆情分析, AI新闻, 算法推荐,
misinformation, deepfake detection,
信息茧房, 舆论传播, AI内容识别,
algorithmic bias, content moderation,
AI治理, platform governance
```

**关注重点**：
- 学术论文（传播学+AI交叉研究）
- 治理政策（各国AI内容监管）
- 平台算法变化（影响内容分发）
- 虚假信息检测技术
- 隐私保护政策

---

## 跨领域通用规则

1. 一条新闻可能同时属于多个领域，取最相关的一个标注
2. 新领域迁移时：替换上述四个领域的关键词表，推理链 R1-R7 全部复用
3. 纯 NLP/CV 基础研究（无直接应用场景）→ 不进入日报
4. 涉及头部厂商（OpenAI/Anthropic/Google/Meta/腾讯/阿里）→ 优先关注
